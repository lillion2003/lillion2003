### ü§ñ 3.2 Modeling Projects

Projects involving the development, training, and comparison of machine learning and deep learning models.

#### üí∞ [Gold Price Forecasting: Time Series vs Machine Learning](https://github.com/lillion2003/gold-price-forecasting)

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![R](https://img.shields.io/badge/R-276DC3?style=flat-square&logo=r&logoColor=white)
![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?style=flat-square&logo=pytorch&logoColor=white)
![XGBoost](https://img.shields.io/badge/XGBoost-337AB7?style=flat-square)

**Description:** Comparative study of traditional time series methods versus machine learning approaches for gold price prediction during the Russia-Ukraine conflict.

**Key Highlights:**

- Comparison of 4 forecasting methods: Holt's Exponential Smoothing, ARIMA, LSTM, and XGBoost
- Multi-step direct forecasting strategy implementation
- Hyperparameter tuning using Rolling Window Cross-Validation
- Performance evaluation using RMSE, MAE, and MAPE metrics

**Tech Stack:** Python (PyTorch, XGBoost, Skforecast), R (forecast, tseries, ggplot2)

#### üáπüá≠ [Thai Sentiment Analysis with WangChanBERTa](https://github.com/lillion2003/thai-sentiment-wangchanberta)

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![Hugging Face](https://img.shields.io/badge/ü§ó_HuggingFace-FFD21E?style=flat-square)
![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?style=flat-square&logo=pytorch&logoColor=white)

**Description:** Enhancing Thai restaurant review sentiment classification using synthetic data augmentation with LLMs (Llama-3.1 and Typhoon 2).

**Key Highlights:**

- Fine-tuning WangChanBERTa for Thai sentiment analysis
- Synthetic data generation using state-of-the-art LLMs
- Comparison of three synthesis approaches: Rewriting, Prompt-based, and Mixed methods
- Achieved 90.99% macro F1-score with the Mixed method (improved from 88.11% baseline)
- Significant improvement in negative class prediction (77.66% ‚Üí 83.01%)

**Tech Stack:** Python, Transformers (Hugging Face), PyTorch, Pandas, Scikit-learn
